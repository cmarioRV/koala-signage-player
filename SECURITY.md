# Security Policy

## Supported Versions

The latest 1.0 release candidate is supported while production acceptance is
in progress. After the stable release, only the latest 1.x version will receive
security fixes.

## Reporting a Vulnerability

Please avoid public disclosure.

Use GitHub Security Advisories when available or contact the maintainer privately.

Include:
- Description
- Reproduction steps
- Impact
- Suggested mitigation (optional)

Thank you for helping improve the project.
