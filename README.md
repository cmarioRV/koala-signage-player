# Koala Signage Player

> A lightweight, offline-first Digital Signage Player written entirely in Swift for Raspberry Pi.

Koala Signage Player is the playback component of the **Koala Signage** ecosystem. It is designed to run on Raspberry Pi devices and provide reliable, unattended multimedia playback for commercial digital signage installations, including **video wall** configurations.

The player is built with a strong focus on reliability, recoverability, and maintainability. It controls the playback engine through **MPV's JSON IPC interface**, allowing seamless playlist updates without restarting the player.

# Design Principles

The player follows a few core principles:

* **Offline-first**

  * Playback must continue even if the server becomes unavailable.

* **Single Responsibility**

  * The player is only responsible for downloading, validating and playing content.

* **Recoverability**

  * Unexpected crashes should automatically recover without user intervention.

* **Deterministic Playback**

  * The currently published playlist should always produce the same playback order.

* **Minimal Dependencies**

  * Leverage the operating system and MPV instead of reimplementing multimedia features.

---

# Running

```bash
.build/release/koala-signage-player \
    --config Resources/config.example.json
```

The player polls its assigned remote manifest without changing local playback. Configure the interval with:

```json
"manifestPollIntervalSeconds": 30
```

At this stage, a newer manifest is decoded and reported in the logs only. Downloads, checksum validation, and remote playlist activation are intentionally not implemented yet.

# About

Koala Signage Player is a personal engineering project focused on exploring native Swift development beyond Apple platforms.

Its primary goals are:

* Build a production-quality Digital Signage player using Swift.
* Explore Linux development with Swift.
* Design a resilient, offline-first architecture.
* Demonstrate software architecture and systems engineering skills through a real-world project.
