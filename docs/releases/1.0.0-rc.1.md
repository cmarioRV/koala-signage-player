# Koala Signage Player 1.0.0-rc.1

This release candidate freezes the Player selected for the first Koala Signage
production acceptance test. It is not the final stable release.

## Included

- Unattended MPV playback managed by systemd.
- Local-first and offline-safe playback.
- Authenticated registration, heartbeat, and manifest polling.
- Staged downloads with expected-size and SHA-256 verification.
- Atomic versioned playlist activation and restart restoration.
- Recurring schedule evaluation, prefetching, fallback, and offline execution.
- Active playlist and media telemetry.
- Safe deployment with configuration preservation and binary rollback.

## Validation completed

- Swift unit and contract tests on macOS and Linux CI.
- Release executable build.
- Raspberry installation and deployment script syntax validation.
- Local Server/Studio production rehearsal through Docker.

## Production acceptance still required

- Deploy to the Raspberry on the final LAN.
- Register against the ThinkCentre Server and enable required device auth.
- Confirm video-wall playback, restart restoration, and offline scheduling.
- Observe uninterrupted operation after a ThinkCentre and Raspberry reboot.

## Deferred beyond v1

- Resumable partial downloads.
- Automatic Player software updates.
- Advanced remote diagnostics.
- Synchronization between independent playback devices.

Promote this candidate to `1.0.0` only after every production acceptance item
in the Server runbook passes.
