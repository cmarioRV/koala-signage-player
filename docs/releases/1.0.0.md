# Koala Signage Player 1.0.0

`1.0.0` is the first stable Koala Signage Player release. It promotes the
functionally unchanged `1.0.0-rc.1` Player candidate after production
acceptance on the Koala 1x3 video wall.

## Stable scope

- Unattended local playback through mpv JSON IPC and systemd.
- Versioned manifest polling, authenticated heartbeats, and current-playback
  telemetry.
- Safe download staging, size and SHA-256 verification, immutable releases,
  and atomic playlist activation.
- Offline-first restoration, cached scheduling, fallback playback, and
  recovery after Server loss or Raspberry restart.
- Recurring schedules with timezone, weekday, cross-midnight, and priority
  evaluation.
- Deployment scripts that preserve configuration and content and roll back a
  failed binary deployment.

## Production acceptance

- Registered against the hardened production Server with a per-device
  credential.
- Downloaded, verified, activated, and reproduced a newly processed playlist.
- Preserved correct color, geometry, continuity, and looping across the
  physical 1x3 wall.
- Activated scheduled content and restored fallback both online and offline.
- Restored the installed remote release after a Raspberry reboot.
- Continued playback through Server and Windows/Docker outages and resumed
  telemetry after recovery.

## Upgrade

Deploy from a clean branch containing the stable tag:

```bash
cd Player
./Scripts/deploy-rpi.sh
```

The deployment updates the reported `appVersion` to `1.0.0` while preserving
the production Server URL, registration token, device state, cached media, and
active release.
