# Contributing

Thank you for your interest in contributing to Koala Signage Player.

## Reporting Issues

Please include:
- Expected behavior
- Actual behavior
- Steps to reproduce
- Raspberry Pi model
- Raspberry Pi OS version
- Swift version

## Pull Requests

1. Open an issue for significant changes.
2. Keep pull requests focused.
3. Follow Swift API Design Guidelines.
4. Update documentation when needed.
5. Ensure the project builds successfully.

## Coding Principles

- Readability first.
- Small, testable components.
- Favor composition over inheritance.
- Keep dependencies minimal.
